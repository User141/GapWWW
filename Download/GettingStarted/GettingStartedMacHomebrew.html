The Basic Steps of a GAP Installation

Below we describe how to compile GAP from source. You might also want to try:

Please note that these alternatives are maintained by third parties and might not provide the latest release and can take a while to be updated.

You can install Gap via Homebrew. To perform a GAP Homebrew installation for the first time, use 

brew install gap-system/gap/gap

To upgrade an existing GAP Homebrew installation, call 

brew upgrade gap-system/gap/gap
