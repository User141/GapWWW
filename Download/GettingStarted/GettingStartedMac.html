The Basic Steps of a GAP Installation

Below we describe how to compile GAP from source. You might also want to try:

Please note that these alternatives are maintained by third parties and might not provide the latest release and can take a while to be updated.

Download

Copy the file to where you want the folder containing GAP to be. For example you can use 

 cp Downloads/gap-4.11.0.zip Projects/

if you downloaded to your Downloads folder and want GAP in your Projects folder.

Then run

 unzip gap-4.11.0.zip

Compiling GAP

To compile GAP you need the Apple developer tools. Install XCode from the Mac App Store.

Then run 

 xcode-select --install

If that does not work log in here with your Apple-ID. Download and install "Command Line Tools for XCode".

Change to the directory gap-4.11.0 (which you just created by unpacking). Then compile GAP. 

cd gap-4.11.0
./configure
make

Both will produce a lot of text output. You should end up with a shell script bin/gap.sh which you can use to start GAP. If you want, you can copy this script later to a directory that is listed in your search path.

bin/gap.sh

Please note that this script must be started from within the Terminal Application. It is not possible to start GAP by clicking this script.
